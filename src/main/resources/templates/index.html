How ResolveIT Works

  1. Citizen files a report using the web portal or mobile app, attaching photos and location.
  2. Report is triaged and routed to the responsible department or officer.
  3. Officer updates status: acknowledged, in progress, resolved. Citizen receives notifications.
  4. Feedback is collected after resolution to improve service quality.
